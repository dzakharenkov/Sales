# Sentry Error Tracking Automation

Автоматизация процесса получения ошибок из системы мониторинга Sentry, их обработки, экспорта в Excel и генерации задач.

## Требования
- Python 3.8+
- Установить зависимости: `pip install -r requirements.txt`

## Установка и запуск
1. Клонируйте репозиторий.
2. Установите зависимости командой: `pip install -r requirements.txt`
3. Убедитесь, что токен Sentry и настройки указаны корректно в скрипте.
4. Запустите скрипт командой: `python sentry_processor.py`

## Описание
Скрипт делает следующее:
- Подключается к Sentry REST API для получения за 14 дней неразрешенных ошибок (макс. 100 на запрос, с пагинацией)
- Извлекает стек-трейс и контекстные данные
- Формирует Markdown файлы с задачами для разработчиков в папку `sentry_tasks/`
- Экспортирует статус задач по проектам в Excel файл `sentry_errors.xlsx`
- Сохраняет резервную `json` копию в файл `sentry_errors.json`
